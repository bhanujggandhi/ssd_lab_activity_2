# SSD Lab Activity 2

#### Instructions to Run:

./q1.sh: Takes argument as a text file using command parameter

If number of lines are odd: (n/2)+1 th line
If number of lines are even: n/2 th line

```sh
$ chmod 744 q1.sh
$ ./q1.sh /path/to/file
```

./q2.sh: Prints all the /usr shells

```sh
$ chmod 744 q2.sh
$ ./q2.sh
```
